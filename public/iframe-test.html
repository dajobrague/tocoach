TopCoach Iframe Embedding Test

Purpose: This page tests if the TopCoach trainer system can be embedded in an iframe (like Go High Level).

Default URL: http://localhost:3000 (change below if needed)

Test Instructions:

  1. Make sure your Next.js dev server is running (npm run dev)
  2. The iframe should load without errors
  3. Try logging in through the iframe
  4. Navigate between pages and verify sessions persist
